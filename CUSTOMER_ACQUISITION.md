### 2. Improve Contact Form (1 hour)
**Add These Fields**:
- Budget range (dropdown: $1K-$5K, $5K-$15K, $15K-$50K, $50K+)
- Project timeline (dropdown: ASAP, 1-3 months, 3-6 months, 6+ months)
- How did you hear about us? (dropdown: Google, Social Media, Referral, Other)
- Project type (checkbox: Website, Web App, Mobile App, Other)

**Why**: Better qualification = better leads = higher conversion

**Where**:
